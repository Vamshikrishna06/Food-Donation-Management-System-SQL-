# Food Donation Management System (SQL-Based)

## Overview

The **Food Donation Management System** is a database-driven project developed using **SQL** to manage food donations efficiently. The system stores information about donors, available food items, NGOs/volunteers, and distribution details, helping reduce food wastage and improve donation tracking.

## Features

* Create and manage donor records
* Store food donation details (food type, quantity, location, date)
* Maintain NGO/volunteer information
* Track food pickup and distribution records
* Retrieve reports using SQL queries

## Technologies Used

* **Database:** MySQL / SQL Server / Oracle
* **Language Used:** SQL (DDL, DML, DQL)

## Database Structure

Typical tables include:

* Donor Table
* Food_Donation Table
* NGO/Volunteer Table
* Distribution Table

## Usage

* Insert new donation details using **INSERT** queries.
* Update donation or pickup information using **UPDATE** queries.
* Remove records using **DELETE** queries.
* Generate reports and insights using **SELECT** queries.

## Applications

* Reduces food wastage
* Efficient tracking of food donations
* Supports NGOs and charity organizations
